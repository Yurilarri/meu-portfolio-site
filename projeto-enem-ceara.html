Análise de Desempenho do ENEM 2024 no Ceará

Um mergulho profundo nos dados educacionais do estado para identificar padrões de desempenho e participação por município.

Objetivo do Projeto

O objetivo principal foi extrair dos dados nacionais do ENEM 2024 um subconjunto focado exclusivamente no estado do Ceará. A partir disso, a análise buscou responder a duas perguntas principais: Quais municípios possuem as maiores notas médias? E quais municípios têm o maior número de participantes?

Metodologia e Ferramentas

A análise foi inteiramente conduzida utilizando a linguagem SAS, aproveitando sua capacidade de manipulação de grandes volumes de dados e seus robustos procedimentos estatísticos.

  1. Filtragem de Dados: Utilizei `PROC SQL` para criar duas novas tabelas (`ENEM_2024_RESULTADOS_CE` e `ENEM_2024_PARTICIPANTES_CE`), contendo apenas os registros onde `SG_UF_PROVA = 'CE'`.
  2. Análise Estatística: Com `PROC MEANS`, calculei as estatísticas descritivas (média, mediana, desvio padrão) das notas para o estado.
  3. Agregação e Ranking: Agreguei os dados por município para calcular a nota média geral e o total de participantes em cada um.
  4. Visualização: Utilizei `PROC SGPLOT` para gerar gráficos de barras, apresentando de forma clara o ranking "Top 10" de municípios para cada uma das métricas analisadas.
